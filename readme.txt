--Application Setup:

1. create a database named "notesDB" in mysql.
2. import the java backend named "backend" in STS or eclipse and run the backend of the system.
3. Open the Angular frontend named "frontend" in VS Code.
4. Open terminal and run commands 
	"npm install"
	"ng serve"
5.Use the application on "http://localhost:4200/"



--Application Functionalities:


The system allows its users to add new notes , edit existing notes , view most recently modified notes and delete notes.

The system also deletes all the old notes of each user except the 10 most recently modified notes.
